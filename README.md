# RISE Kernel and Virtualization WG

## About RISE

[RISE](https://riseproject.dev) is a collaborative, industry-led initiative under the Linux Foundation that accelerates open-source software development for the RISC-V architecture.

## Charter

This working group covers the Linux kernel, KVM and other hypervisors, as well as debuggers (e.g. lldb), profilers (valgrind), and similar tools for RISC-V. Our objectives are to:

- Help collaboration across companies to avoid duplication of work
- Accelerate RISC-V kernel, virtualization, and tooling development and upstreaming
- Communicate on the readiness of the respective ecosystems
